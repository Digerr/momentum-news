# 📰 MOMENTUM — Новости в движении

Современный новостной портал с реальными RSS-лентами из 6 источников. **Single-file PWA на vanilla JS** — без фреймворков, без зависимостей, без бэкенда.

🔗 **Live:** https://digerr.github.io/momentum-news/

## ✨ Возможности

### 📡 Реальные данные
- **6 источников:** Lenta.ru, ТАСС, РИА Новости, Interfax, Коммерсантъ, Хабр
- **RSS-парсинг** через CORS-прокси (`corsproxy.io` + `allorigins.win` как fallback)
- **Автообновление** каждые 5 минут
- **~150 статей** в реальном времени

### 🌤️ Тикер с данными
- **Погода Москвы** (Open-Meteo API)
- **Курсы валют** USD/EUR/CNY (ЦБ РФ)
- Дата и день недели

### ⚡ Интерактив
- **Cmd/Ctrl+K** — командная палитра с поиском по новостям
- **Закладки** — drawer справа, кнопка на карточке, счётчик в шапке
- **История просмотров** — блок «Вы читали»
- **Фильтр по источникам** — popover с чекбоксами
- **8 категорий** с живыми счётчиками
- **Поиск по тегам** из сайдбара

### 📖 Чтение статьи
- **Полный текст** статьи подгружается прямо в модалку (через CORS-прокси)
- **Reading progress bar** сверху — показывает насколько прочитано
- **Web Share API** — нативный шеринг с телефона
- **AI-конспект** — extractive summary из 3 ключевых предложений
- **Кэш полного текста** в localStorage (последние 50 статей)
- **Кнопка «Читать оригинал»** — ссылка на источник

### 🎨 UI/UX
- **Тёмная/светлая тема** с сохранением выбора
- **Мультиязычность** RU/EN — переводится весь UI
- **Mobile-first** адаптивность
- **Sticky-шапка** с тикером
- **LIVE-индикатор** с пульсирующей точкой
- **Framer-Motion-стиль анимации** (на CSS)

### 📱 PWA
- **Service Worker** — кэш для оффлайна (network-first стратегия)
- **Install prompt** — баннер «Установить MOMENTUM» через 5 сек
- **Theme-color** для status bar
- Можно установить как приложение на телефон/десктоп

### 💾 Persist (localStorage)
- Закладки (до 200 статей)
- История просмотров
- Выбранная тема
- Язык интерфейса
- Выбранные источники
- Кэш полных текстов (последние 50)

## 🚀 Деплой

Файл `momentum.html` — это **один автономный HTML-файл** со всем кодом внутри (HTML + CSS + JS). Его можно:

- Открыть двойным кликом (но real RSS требует http:// или https://)
- Залить на любой статический хостинг (GitHub Pages, Netlify, Vercel, Cloudflare Pages)
- Отправить друзьям в Telegram/WhatsApp

Текущая версия задеплоена на **GitHub Pages**:
- Репозиторий: https://github.com/Digerr/momentum-news
- Live: https://digerr.github.io/momentum-news/

## 🛠 Технологии

- **Vanilla JS** (ES2020+, без фреймворков)
- **CSS Custom Properties** для тем
- **DOMParser** для парсинга RSS/XML
- **fetch + AbortSignal.timeout** для запросов
- **localStorage** для persist
- **Service Worker API** для PWA
- **Web Share API** для шеринга
- **beforeinstallprompt** для install banner

## 📊 Источники и парсеры

| Источник | Метод | Полный текст? |
|---|---|---|
| Lenta.ru | JSON-LD `text` поле | ✅ |
| РИА Новости | `.article__text` дивы | ✅ |
| Коммерсантъ | `.article_text` + JSON-LD | ✅ |
| Interfax | `<p>` теги | ✅ |
| ТАСС | anti-bot блокировка | ⚠️ excerpt + ссылка |
| Хабр | anti-bot блокировка | ⚠️ excerpt + ссылка |

## 🔧 CORS-прокси

Браузер не может напрямую обращаться к RSS-лентам (CORS-политика). Используются прокси:
1. `https://corsproxy.io/?url=<encoded-url>` — основной
2. `https://api.allorigins.win/raw?url=<encoded-url>` — fallback
